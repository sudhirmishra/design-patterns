# Design Patterns 

The repository contains implementation of all the Design Patterns listed below:

1. Observer Pattern
2. Factory Pattern
3. Abstract Factory Pattern
4. Chain of Responsibility Pattern
5. Decorator Pattern
6. Bridge Pattern
7. Adapter Pattern
